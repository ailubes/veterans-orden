## 1. Design Philosophy

### Timber Design System

The app follows the **Timber** brutalist architecture design system:

- **Exposed Structure**: Grid lines, beams, and joints are visible design elements
- **Raw Materials**: Warm paper tones, dark timber colors, terracotta accents
- **Functional Aesthetics**: Form follows function with no unnecessary decoration
- **Bold Typography**: Strong headlines with technical body text
- **Grain Texture**: Subtle paper texture overlay for authenticity

### Core Principles

| Principle | Implementation |
|-----------|----------------|
| **Structural Honesty** | Show grid lines, borders, and construction elements |
| **Material Warmth** | Use warm paper backgrounds, not pure white |
| **Bold Contrast** | High contrast text for accessibility |
| **Functional Animation** | Purposeful micro-interactions, not decorative |
| **Ukrainian Identity** | All UI text in Ukrainian, uk-UA locale formatting |
